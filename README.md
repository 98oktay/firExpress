# FirExpress
Express hosting for firebase functions
## install
Download this repository and
### 1. Setup Firebase

```
npm install -g firebase-tools
```
Login to Firebase CLI
```
firebsae login
```
### 2. Set your project

```
firebase list (Show your project Id's)
```
and 
```
firebase use <Project-Id>
```

### 3. Install NPM packages

```
npm install 
```
### 4. Local Development
```
npm run dev
```
